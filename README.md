# Scented by DDSD

A luxury e-commerce storefront built with React, TypeScript, Tailwind CSS, and Bun.

## Getting Started

1. **Install dependencies**:
   ```bash
   cd scented-by-ddsd
   bun install
   ```

2. **Run development server**:
   ```bash
   bun dev
   ```

3. **Build for production**:
   ```bash
   bun run build
   ```

## Development

- **Linting**: `bun run lint`
- **Testing**: `bun run test`
- **Formatting**: `bun run format`

## Environment Variables

Copy `.env.example` to `.env` and fill in your values.

## Color System

- Primary Gold: `#B7852B`
- Background Black: `#0B0B0B`
- Accent White: `#FFFFFF`

## Admin Access

Default (Mock) Admin Password: `changeme` (See `.env.example`)
